Understand the code

When declaring a plugin, the syntax is new sigplot_plugins.[plugin name].

To add a plugin to the plot, use the syntax plot.add_pluin([plugin_name]).

For the annotation plugin, a variable called annotations is declared, and the syntax annotations.add_annotation is used to fill the annotations variable with the all of the annotations that will be present on the plot.

The add_annotation method uses key-value pairs to initialize the annotation. For the text annotation, the key-value pairs are x: 20 (x coordinate), y: 0 (y coordinate), value: "Text" (What will be shown on the plot), color: "#FA7B92", highlight_color: "green" (color on hover), and popup: "[Pop-up text]".

The image annotation is declared in the image element's onload function and does not have a color or highlight_color listed. Its value is set to the image element whose source is set to the path to the image.
