Understand the code

In this example, the plot has a listener for the a double-click and an event called "mtag". More information about mtag events can be found in the mouse events section of the Developer Tips page.

The plot adds a highlight using the x value (x coordinate of beginning of rubberbox) and w value (width of the rubber box) of the mtag event. Then, when the middle-click event is heard, if the x value of the event is within the range of a highlight, that highlight is deleted.

A dom element element called 'clear' has an onlick function that will remove all highlights.
