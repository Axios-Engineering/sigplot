During instantiation

In this example, the readout is set to be hidden when the mouse is not on the plot, the panbars are set to be hidden when the plot is not zoomed in, the background of the grid is set to a gradient (although it could be set to a solid color as well using a hexcode or the name of any basic color), and the background and foreground colors are inverted (xi: true). The default is to have a black background with a white foreground.

change_settings

In this example, the change_settings function is used to change the appearance of the plot after instantiation. Please see the Functions section of the Developer's Tips page for documentation of the arguments than can be used in this function.

Using buttons

To create a button like "invert" that modifies the plot, use an onclick event that calls change_settings on the plot. You wil need an html dom element to make an onclick event. Click the "HTML" and "CSS" tabs to see the dom element and css used for the invert button.
