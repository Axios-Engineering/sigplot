Understand the code

When declaring a plugin, the syntax is new sigplot_plugins.[plugin name].

To add a plugin to a plot, use the syntax plot.add_plugin([plugin name]).

For the boxes plugin, a variable called boxes is declared to hold all of the boxes that will go on the plot.

The boxes are declared and added to the boxes variable using the boxes.add_box method. Each box has attributes x (x coordinate), y (y coordinate), w (width), h (height), text, and fill (true or false).
