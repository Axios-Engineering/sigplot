Understand the Code

An event listener has been added to the plot. It listens for an event called 'mtag' and performs a nameless function that takes in the event itself as its argument. It first print the values stored in the plot for x and y in both Mx structure and the Gx structure. Then it changes the values of the dom elements under the plot to the x and y value associated with the mtag event.

Click the "JavaScript" button on the top of the example to hide the JavaScript and click the "Console" button to show the console. Click anywhere on the plot to see the Mx values for y and x printed on the first line of the console and the Gx values for y and x printed on the second line of the console. You will see that the x and y values associated with the mtag match the x and y values stored in the Gx structure. This is because both the Gx structure and the mtag event are showing the x and y values in terms of the y and x axis labels, while the Mx structure stores the values as postitions on the canvas as a whole, not just the plot.
