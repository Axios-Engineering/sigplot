Understand the code

The function update_rtraster creates a line of horizontal data that is pushed to the plot using plot.push. If the the plot layer does not exist yet (the first time that the function is called), then the overlay_pipe method is used to create the layer.

Experiment with the data

The framesize refers to the length horizontally of the data. Try modifying this value to see how it affects the plot.

Change the Math.random function to something different, like Math.cos for example, and see how it affects the data.
