Understand the Code

To create a raster plot, set the variable that will hold your plot to a new plot using new sigplot.Plot(document.getElementById('plot')). Just like a regular plot, the div or dom element that will hold the plot must be specified.

To plot raster data, overlay href can be used. Simply include the url or relative path to the file you would like to plot.
