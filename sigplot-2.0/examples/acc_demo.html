Understand the Code

When declaring a plugin, the syntax is new sigplot_plugins.[plugin name].

For the accordion plugin, this declaration is set will key-value pairs that describe the attributes of the accordion. In this example, the pairs are draw_center_line: true, shade_area: true, draw_edge_lines: true, direction: ["vertical" or "horizontal"], and edge_line_style: {strokeStyle: [some hex-color-code]}

To add the plugin to the plot, the following syntax is used: plot.add_plugin([var that holds plugin], 1);

To set the center and width of the accordion, the following lines of code can be used: plot.set_center([center value]); and plot.set_width([width value]);.
