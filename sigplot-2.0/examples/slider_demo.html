Slider Plugin

JS Bin on jsbin.com

Understand the Code

When declaring a plugin, the syntax is new sigplot_plugins.[plugin name].

To add a plugin to a plot, use the syntax plot.add_plugin([plugin name]).

For the slider plugin, a variable is created for each slider, one called slider1 and one called slider2. The only key-value pair inlcuded when declaring the slider is name: "[Slider name]". Later in the code, the two sliders are paired with slider1.pair(slider2); and slider2.pair(slider1);. The slider starting positions are set in a similar manner with the syntax slider.set_position([start position]).

Sliders emit "slidertag" events when they are moved programatically or by the user and only emit "sliderdrag" events when they are moved by the user. Listeners can be added to the plot for these events.

Experiment with the Code

Add another slider to the plot that is not pair to any other sliders.

In the curly brackets for the callback function of one of the event listeners, write code that modifies the plot and see how the plot responds when the sliders are moved.
