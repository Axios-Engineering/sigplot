Understand the Code

The plot is instantiated first and then updated with the function update_rtiq. The update function calls overlay_pipe to declare the data layer (if not already done), and then plot.push(0, [data]); is used to update the data in the plot.

Note that the cmode variable is set to 5, which is equivalent to "real/imag" and "imag/real".
