Real-Time Plotting

JS Bin on jsbin.com

The update_rtplot function updates the plot by using the reload method. If the data layer does not exist (the function is being called for the first time), then then overlay_array is used to create the layer that will contain the data.

The Math.random function is being used to fill a JavaScript array that will be reloaded into the plot.

Experiment with the data

In the for loop, where i <= 1000, change 1000 to a different number to see how the x values of the plot change.

In the window.setInterval function, change the value of 500 to see the plot update more quickly or slowly.
