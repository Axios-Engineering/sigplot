Custom Menu

JS Bin on jsbin.com

Understand the Code

In this example, the plot has the menu turned off on instantiation with the key-value pair nomenu: true. If you scroll to the bottom of the JavaScript, you will see that the plot has listeners for the "showmenu" and "hidemenu" events. These are used to display or hide the dom elements that will be the custom menu.

Each of the buttons in the custom menu has a unique id in the html, and an onclick event in the JavaScript.

Note that the menu is displayed on the middle-click and is hidden when a selection is made or when the mouse is clicked elsewhere, exactly copying the functionality of the built-in menu.

Experiment with the Functionality

You can change the functionality of the onclick functions to modify the plot in different ways.
