Plotting Audio

JS Bin on jsbin.com

Understand the Code

Once the plots are intstantiated, an audio context and audio are declared if the browser supports it.

If possible, the function loadPlotFromWav will be called which uses a get request to directly load the .wav file then shortens the data to make it more manageable for the browswer. If not, then the function loadPlotFromHref is called, which calls overlay_href.

A slider is created to swipe along the plot as the audio is played.

This plot uses the PlaybackControlsPlugin that listens for when the audio is playing and updates the buffer and start times accordingly.

The code then checks if the audio is ended, and if it is, it will be paused.

The slider is updated using a function called update_slider as the data is played.

If the broswer supports creating an analyser, then one is created and the analyze function is called in a set interval.
